AI-Based Interview Coach

Technologies: Frontend: HTML, CSS, JavaScript | Backend: Python, Flask | APIs: Google Gemini API | Others: GitHub, SQLite (optional)

Description: The AI-Based Interview Coach is an intelligent web application designed to help users prepare for job interviews by simulating real-time interview experiences. The system uses speech-to-text, Google Gemini API, and Flask to provide dynamic, interactive coaching and feedback.

  • AI-Powered Questioning: Uses Google Gemini API to ask follow-up questions intelligently based on user responses.
  • Smart Feedback & Summary: Provides a concise summary of the interview session, including areas of improvement.
  • Web Interface with Flask: Clean, user-friendly interface built using Python Flask framework.

Purpose: To help students and job seekers build confidence, improve their interview skills, and receive AI-powered feedback anytime, anywhere.

Technologies: HTML, CSS, JavaScript

Description: The YouTube Video Downloader is a Python-based application that allows users to download YouTube videos by simply pasting the video URL. Purpose: To enable users to download YouTube content for offline use, especially for students or professionals who want to save learning videos, tutorials, or songs without relying on internet access.
